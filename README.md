# amazon-sales-analysis(Excel Project)
## Overview
This project analyzes Amazon sales data using Excel. It explores order trends, cancellations, delivery performance, and regional sales targets through pivot tables and KPIs.

## Skills Demonstrated
- Excel PivotTables & charts
- Delivery time calculations
- Regional sales comparisons
- Business insights & recommendations

## Key Findings
- North region achieved the highest sales but had frequent cancellations.
- East region showed the longest average delivery time (8.1 days).
- Fashion category had the highest cancellation rate.

## Business Recommendations
- Improve delivery performance in East region.
- Review cancellation patterns in Fashion category.
- Optimize fulfillment partner allocation.
